GET /staticpage

This is the view for the "GET /staticpage" route, a static HTML page.

This page simply demonstrates, that you can also use a static HTML page as a view.

Beware!

The links in the menu might not work, depending on your base path or server configuration. There is no way to determine this on a static page. Use your browser's back button to get back to where you came from.
